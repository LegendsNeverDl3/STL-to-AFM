# Implementation Summary: Acoustic Levitation Simulator v2.2

The following changes have been implemented to improve the physical accuracy, visual feedback, and user experience of the acoustic levitation mapping tool.

## 1. Physics & Force Vectorization
- **Net Force Calculation:** Confirmed and explicitly implemented the global free-body force vector as: 
  `Net Force = Σ (Local Surface Radiation Forces) + (Global Gravity)`.
- **Force Breakdown:** Added a dedicated section in the stats panel to show **Acoustic Lift (Z)** and **Gravitational Weight (Z)** independently. This allows the user to see exactly how much "lift" is being generated by the sound field in Newtons.
- **Unit System:** Verified the consistent use of the `mm-g-s` unit system for gravity and acoustic radiation pressure ($1\text{ Pa} \cdot \text{mm}^2 = 1\text{ g} \cdot \text{mm/s}^2$).

## 2. Advanced Visual Scaling
- **Logarithmic Force Arrows:** Resolved the "Visual Saturation" issue where the orange Net Force arrow appeared static in size due to extreme magnitudes.
- **Log-Scaling Logic:** Transitioned the Net Force magnitude from linear/capped to **Logarithmic** using `log1p(|F|/G)`.
  - **Result:** The orange arrow now visibly grows and shrinks across thousands of magnitudes, providing a "soft landing" as the object approaches equilibrium.
- **Binary Directional Hiding:** Added logic to completely hide the arrow when the net force is within a microscopic epsilon of zero, indicating **Stable Levitation**.

## 3. UI/UX Overhaul (Command Dashboard)
- **High-Contrast Dark Theme:** Implemented a "Premium Dark" aesthetic using deep indigo/charcoal backgrounds (`#1a1c2c`) and neon-cyan accents (`#00e0ff`). This resolves previous contrast issues against the radial-gradient backdrop.
- **Accordion-style Sidebar:** Replaced the long, scrolling sidebar with a collapsible **Bootstrap Accordion**.
  - **Category 1: Environment Setup:** Model, Material, Phase, and Power.
  - **Category 2: Visualization:** Toggles for various field and force arrows.
  - **Category 3 & 4: 6-DOF Movement:** High-precision sliders for translation and rotation.
- **Three-Column layout (Self-Adjusting):** Reorganized the layout so that Column 1 & 2 handle setup/movement, while Column 3 (the Graph) remains **Fixed/Sticky** on the right side of the viewport.
- **Elimination of Scrolling:** Vertical height is now managed so the entire tool fits on a 1080p screen without scrolling.

## 4. Precision Control Tools
- **Sound Power Filter (1-100%):** Added a new slider that acts as a global amplitude multiplier. 
  - **Utility:** Lowering the power "softens" the acoustic field's stiffness, broadening the levitation window and making it easier to hit the zero-point with manual slider steps.
- **Micro-Slider Step (0.1 mm):** Refined the Z-axis and XY sliders to $0.1\text{ mm}$ steps for precise node-finding.
- **Dynamic Mass Display:** Added a dynamic calculation of the object's mass in **Grams (g)** and **Milligrams (mg)** to the bottom information panel.

## 5. Compatibility Fixes
- **DBC Versioning Patch:** Resolved a `TypeError` related to the `bs_size` keyword by transitioning to the standard `size` argument compatible with current `dash-bootstrap-components` (v1.x+).
- **Callback Signature Sync:** Updated the `update_physics` callback to handle the new `sound-power` input and correctly scale both Simplified and Gorkov force models.
